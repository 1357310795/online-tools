# 💬 概述

一款简洁高效的在线工具箱｜JSON 在线解析及格式化验证、Base64 编解码、URL 编解码

- [x] JSON 在线解析及格式化验证
- [x] Base64 编解码
- [x] URL 编解码
- [ ] Crontab 执行时间计算
- [ ] RGB 颜色值转换

## 🚀 本地运行

1. 克隆仓库：

```sh
git clone https://github.com/sunls24/online-tools
```

2. 安装依赖项：

```bash
pnpm install
```

3. 本地运行：

```bash
pnpm dev
```
